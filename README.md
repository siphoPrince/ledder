# **Ledder — Smart Job Tracking Made Simple**

> _Organize your career journey with clarity and focus._  
> Ledder helps you track applications, manage job statuses, and stay motivated throughout your job hunting journey.

---

## Overview

**Ledder** is a modern **Job Tracking Web App** designed to help users stay on top of their job applications.  
It provides a clean, user-friendly dashboard where you can track job titles, companies, application statuses, interview stages, and more all in one place.  

Built with **React**, **Node.js**, and **SQLite**, Ledder combines simplicity with functionality, helping job seekers stay organized and confident in their search.

## Tech Stack

| Category | Tools Used |
|-----------|------------|
| **Frontend** | React (Vite) ⚡ |
| **Styling** | Tailwind CSS |
| **Backend** | Node.js + Express.js |
| **Database** | SQLite3 |
| **Auth** | Passport.js with Sessions 🔐 |
| **Deployment** | Vercel (Frontend) + Render / Railway (Backend) |

---

## 💡 Features

✅ Create, update, and delete job applications  
✅ Track application status (Applied, Interviewing, Offer, Rejected)  
✅ Visualize your progress using charts and statistics  
✅ User authentication with sessions  
✅ Responsive, modern UI with Framer Motion transitions  
✅ Local SQLite storage for lightweight persistence

## Setup & Installation

# Clone the repository
git clone https://github.com/<siphoprince>/ledder.git
cd ledder

# Install backend dependencies
cd server
npm install

# Start backend
npm run dev

# In a new terminal, run frontend
cd ../client
npm install
npm run dev
